name: corporate-research
description: |
  公司贷款申请时，对申请企业进行结构化财务与运营风险分析。
  使用 rag_retrieve 工具，针对“财务风险”和“运营风险”两个维度各执行 3 次有针对性的检索，
  然后按照指定模板撰写简洁、数据驱动的信贷背调报告（400-500字，3段式结构）。
---

研究方法论（严格执行以下步骤）

1. 财务风险分析（必须执行以下 至少1 次 rag_retrieve 查询）
   - 查询1：核心偿债能力指标（资产负债率、流动比率、速动比率、利息保障倍数 2023-2024）
   - 查询2：现金流质量与可持续性（经营现金流净额、自由现金流、现金流与净利润匹配度 2023-2024）
   - 查询3：盈利能力与趋势（毛利率、净利率、ROE、ROA 近三年变化及原因）

2. 运营风险分析（必须执行以下 至少1 次 rag_retrieve 查询）
   - 查询1：主营业务收入结构与集中度（按产品/行业/客户/地区分拆 2024年占比及变化）
   - 查询2：主要经营风险点（存货周转率、应收账款周转率、供应链依赖、价格波动风险）
   - 查询3：关联交易与对外担保规模及异常情况（2024年关联方交易金额、性质、担保余额）

3. 报告撰写要求
   - 总字数：400–500 字（中文）
   - 语气：客观、专业、审慎、数据驱动
   - 结构：严格三段式
     第一段：财务风险分析（引用关键指标及趋势，指出主要问题）
     第二段：运营风险分析（引用业务集中度、周转效率、关联交易等关键点）
     第三段：综合判断 + 贷款发放建议（明确倾向：可适度支持 / 需严格控制额度与担保 / 暂不宜介入 / 存在重大风险需进一步核实）
   - 所有重要数据必须标注来源（chunk_id 或 页码+类型）
   - 避免空话、套话、过度推测
